# Yuma UI

Biblioteca moderna de componentes React com tema dark cyberpunk. Copy & Paste components.

## Stack

- React 18
- TypeScript
- Tailwind CSS v4
- Vite
- Motion (Framer Motion)
- Lucide React (Icons)

## Desenvolvimento

```bash
# Instalar dependências
npm install

# Rodar em desenvolvimento
npm run dev

# Build para produção
npm run build

# Preview do build
npm run preview
```

## Uso

Copie e cole os componentes diretamente no seu projeto. Não é necessário instalar como pacote.

## Componentes Disponíveis

- **Button** - Botão com variantes primary, secondary, outline, ghost
- **Card** - Card com glass morphism e efeitos glow
- **Badge** - Badge com animação opcional
- **Input** - Input com estado de erro
- **Container** - Container responsivo
